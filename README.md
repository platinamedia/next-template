# next-template

## Установка

### [degit](https://github.com/Rich-Harris/degit)

```bash
npx degit platinamedia/next-template ./path/to/my-app
```

```bash
npm i
```

### Установка `tailwindcss`

```bash
node scripts/addTailwind.mjs
```

## Запуск ДЕВ-окружения

```bash
npm run dev
```

## Сборка для ПРОДа

```bash
npm run build
```
